# Welcome To Siddhiworld - Social Media Application

## Overview
A social media platform combining Instagram-style photo sharing with YouTube-style short video content, featuring user profiles, content feeds, and messaging capabilities.

## Authentication
- Internet Identity integration for user sign up, login, and logout
- Secure user session management

## User Profiles
- Profile creation with username and bio
- Profile picture upload functionality
- Personal post gallery showing user's uploaded content
- View own profile and other users' profiles

## Content Creation & Management
- Photo upload with caption support
- Short video upload (similar to YouTube Shorts format)
- Content stored in backend with metadata (upload time, user ID, captions)
- Support for common image formats (JPEG, PNG) and video formats (MP4)

## Feed System
- Main feed displaying all posts from all users
- Posts ordered chronologically by upload time (newest first)
- Mixed content display (photos and short videos)
- Infinite scroll or pagination for content loading

## Content Interactions
- Like functionality for posts (users can like/unlike)
- Comment system allowing text comments on posts
- Like and comment counts displayed on posts
- Backend storage of all interactions with user associations

## Short Videos Section
- Dedicated section for browsing short video content
- Autoplay functionality for videos
- Swipe/scroll navigation between videos
- Full-screen or optimized viewing experience

## Explore Page
- Trending content discovery based on engagement metrics
- Algorithm considering likes and comments to surface popular content
- Grid or feed layout for content browsing

## Messaging System
- One-to-one text messaging between users
- Chat history storage in backend
- Real-time or near real-time message delivery
- Message threads organized by conversation partners

## Data Storage (Backend)
- User profiles and authentication data
- All uploaded content (photos and videos) with metadata
- Post interactions (likes, comments) with user associations
- Chat messages and conversation threads
- User relationships and engagement metrics for trending algorithm

## User Interface
- Clean, modern design inspired by Instagram and YouTube
- Responsive layout optimized for both mobile and desktop
- Minimal interface with intuitive navigation
- Bottom navigation bar for mobile (Feed, Explore, Create, Profile)
- Header navigation for desktop with search and user menu

## Core User Flows
- User registration and profile setup
- Content upload and sharing
- Feed browsing and content interaction
- Short video consumption with swipe navigation
- Direct messaging with other users
- Profile management and content viewing
